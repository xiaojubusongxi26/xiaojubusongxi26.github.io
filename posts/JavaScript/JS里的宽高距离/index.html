JS 里的那些宽高距离

小橘不颂兮2024-04-122024-04-12

屏幕尺寸

window.screen指向 Screen 对象，其属性包括：

  • height： 返回屏幕的总高度

  • width： 返回屏幕的总宽度

  • availHeight：屏幕高度（不包括 window 任务栏）

  • availWidth：返回屏幕宽度（不包括 window 任务栏）

  • availTop：屏幕视口顶端距离屏幕顶部的距离

  • availLeft：屏幕视口左端距离屏幕左边的距离

  • colorDepth：返回目标设备或缓冲器上的调色板的比特深度

  • pixelDepth：返回屏幕的颜色分辨率（每像素的位数）

屏幕距离

window 对象下：

  • screenTop(screenY)：浏览器窗口左上角到屏幕上边缘的距离
  • screenLeft(screenX)：浏览器窗口左上角到屏幕左边缘的距离。

⚠️ 注意事项：

浏览器窗口

window 对象下：

  • outerWidth：浏览器窗口的宽度
  • outerHeight：浏览器窗口的高度

⚠️ 注意事项：

浏览器视口

浏览器内容窗口的宽高（不包含控制台部分）：

  • window.innerWidth
  • window.innerHeight

元素宽高

只读属性，存在于任何元素上，对于内敛元素以及没有 CSS 样式的元素为 0；否则，它是元素内部的宽度（以像素为单位）。该属性包括内边距（padding），但不包括边框（border）、外边距（margin）和垂直滚动条（如果存在）。

（来自 MDN）备注： 该属性值会被四舍五入为一个整数。如果你需要一个小数值，可使用 element.getBoundingClientRect()。

  • Element.clientWidth
  • Element.clientHeight

网页大小

  • Element.scrollWidth
  • Element.scrollHeight

只读属性，包括由于 overflow 溢出而在屏幕上不可见的内容。

scrollWidth值等于元素在不使用水平滚动条的情况下适合视口中的所有内容所需的最小宽度。宽度的测量方式与clientWidth相同：它包含元素的内边距，但不包括边框，外边距或垂直滚动条（如果存在）。它还可以包括伪元素的宽度，例如::before或::after。如果元素的内容可以适合而不需要水平滚动条，则其scrollWidth等于clientWidth

⚠️ 注意事项：

滚动距离

  • Element.scrollLeft：元素最左端和窗口中可见内容的最左端之间的距离。即当前左滚的距离

  • Element.scrollTop：元素最顶端和窗口中可见内容的最顶端之间的距离。即当前上滚的距离

偏移量

  • Element.offsetWidth：元素宽 + padding + border + 滚动条

  • Element.offsetHeight：元素高 + padding + border + 滚动条

  • Element.offsetLeft：当前元素内容区域（包括 border）左边缘到 offsetParent 内容区域（不包括 border）左边缘的距离。

  • Element.offsetTop：当前元素内容区域（包括 border）顶部到 offsetParent 内容区域（不包括 border）顶部的距离。

  • Element.offsetParent：是一个只读属性，返回一个指向最近的（closest，指包含层级上的最近）包含该元素的定位元素。如果没有定位的元素，则offsetParent为最近的table,table cell或根元素（标准模式下为html；quirks 模式下为body）。当元素的style.display设置为”none”时，offsetParent返回null。offsetParent很有用，因为offsetTop和offsetLeft都是相对于其内边距边界的

相关情况：

  1. 元素自身有 fixed 定位，父元素不存在定位，则 offsetParent 的结果为 null（firefox 中为：body，其他浏览器返回为 null）
  2. 元素自身无 fixed 定位，且父元素也不存在定位，offsetParent 为元素
  3. 元素自身无 fixed 定位，且父元素存在定位，offsetParent 为离自身最近且经过定位的父元素
  4. 当元素的 style.display 设置为 “none” 时，offsetParent 返回 null。
  5. body 的 offsetParent 是 null
